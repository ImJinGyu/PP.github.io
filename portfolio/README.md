# ImJinGyu Portfolio

정적 HTML/CSS/JS 기반 개인 포트폴리오 사이트입니다.

## 이번 개편 내용
- 개발자 소개 문구를 실무 경험 중심으로 재작성
- 기술 스택을 카테고리별로 정리
- 경력/교육/자격 내용을 채용 담당자가 보기 좋은 형태로 재구성
- 실무 프로젝트 7건을 역할/기술/성과 기준으로 요약
- 대표 프로젝트(부산온나) 상세 페이지를 기술 중심으로 보강
- Contact / GitHub / 핵심 강점 섹션 추가

## 주요 파일
- `index.html` : 메인 포트폴리오 페이지
- `portfolio-details.html` : 대표 프로젝트 상세 페이지
- `assets/css/style.css` : 커스텀 스타일 포함

## 배포
GitHub Pages 또는 일반 정적 호스팅에 그대로 업로드하여 배포할 수 있습니다.
